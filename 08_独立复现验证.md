# Week6交付包独立复现验证

## 验证目标

确认导师只取得`Week6_AB监控预测交付包`文件夹时，无需相邻Week5目录或上级`support/`目录，即可运行Agent抽样回放、全量回放和离线预测。

## 包内依赖

| 路径 | 用途 |
|---|---|
| `runtime/agent_workflow_mock.py` | 全量和抽样回放使用的本地Agent Mock |
| `runtime/build_raw_table_replay.py` | 六类场景抽样回放 |
| `support/intervention_psm_did_results.csv` | 离线预测使用的PSM-DID效应输入 |
| `synthetic_data/generated/` | 不含真实用户的合成五表 |

上述Python脚本只使用标准库。

## 隔离验证

验证日期：2026-07-23。

将Week6文件夹复制到新的临时目录，确保其父目录不存在Week5运行时和外部效应文件后，执行：

```bash
python3 runtime/build_raw_table_replay.py \
  --data-dir synthetic_data/generated \
  --output-dir /tmp/week6_raw_replay

python3 synthetic_data/run_full_agent_replay.py \
  --data-dir synthetic_data/generated \
  --output-dir /tmp/week6_full_replay \
  --dashboard-data /tmp/week6_agent_test_data.js

python3 offline_simulation/build_level2_forecast.py \
  --data-dir synthetic_data/generated \
  --output-dir /tmp/week6_forecast
```
